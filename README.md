# Chronic Kidney Disease Predictor

A machine learning project that predicts Chronic Kidney Disease (CKD) using 4 different classification models.

## Models Used
- Decision Tree
- Naive Bayes
- K-Nearest Neighbors (KNN)
- Artificial Neural Network (ANN)

## Dataset
[CKD Disease Dataset – Kaggle](https://www.kaggle.com/datasets/mansoordaku/ckdisease)

## Steps
1. Data loading & exploration
2. Data cleaning & preprocessing
3. Label encoding & feature scaling
4. Model training & evaluation
5. Accuracy, Precision, Recall, F1-Score comparison

## Tools
Python | Pandas | Scikit-learn | TensorFlow/Keras | Matplotlib | Seaborn
